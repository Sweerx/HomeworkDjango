Bootstrap

Bootstrap — популярный фреймворк для создания сайтов и веб-приложений с открытым исходным кодом. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения. Bootstrap использует современные наработки в области CSS, JavaScript и HTML, поэтому использование данного инструмента не всегда может быть правильным решением при использовании старых браузеров.
